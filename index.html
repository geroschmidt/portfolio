Sobre mi

Nací en el año 1999, en Mendoza, Argentina. Realicé mis estudios en la Universidad del Aconcagua, estudiando Licenciatura en Informatica y Desarrollo de Software. Me considero una persona creativa, con un fuerte interés en el trabajo en equipo y con sólidas bases en diversas tecnologías, lo que me permite ofrecer diferentes soluciones y perspectivas a la hora de abordar un problema.

A lo largo de mis estudios he adquirido habilidades en diversas tecnologías, incluyendo C++, C#, Java, PHP, JavaScript, Swift, Git, MySQL, SQL, Spring Boot y Docker, entre otras. He tenido la oportunidad de poner en práctica algunos de mis conocimientos trabajando como desarrollador independiente.

Todo Oficio

Formé parte de un proyecto de migración de plataforma, donde nuestro objetivo era trasladar una plataforma basada en WordPress a una solución web personalizada diseñada específicamente para cumplir con las necesidades de nuestro cliente. Mi papel principal en el proyecto consistió en recopilar los requisitos del cliente, distribuir las tareas de desarrollo, diseñar e implementar interfaces de usuario clave, como el inicio de sesión, y configurar un sistema de gestión de alertas.

CM BIKE

Fui responsable del desarrollo de un software para la gestión comercial de una tienda de bicicletas, con el objetivo de optimizar procesos y mejorar la eficiencia del negocio. El software que desarrollé permitió la gestión de clientes, productos, ventas, servicios técnicos y las marcas con las que trabajaban. Durante el desarrollo del proyecto, utilicé tecnologías como HTML, CSS, PHP, JS y SQL para crear una interfaz de usuario intuitiva y funcional, así como una base de datos sólida y segura. Como resultado, el software que desarrollé permitió una gestión más eficiente de la tienda de bicicletas, aumentando la productividad y reduciendo errores y tiempos de inactividad. Fue un proyecto desafiante y gratificante que me permitió aplicar mis habilidades de programación y trabajar en un entorno colaborativo y dinámico.

POKEDEX

En este proyecto, decidí desarrollar una pequeña Pokédex de forma nativa para iOS con Swift, pude aplicar mis conocimientos sobre el consumo de API REST. Para facilitar el proceso de consumo de la API, opté por utilizar la librería Alamofire, la cual instalé mediante CocoaPods. Otro punto desafiante era lograr que la aplicación fuese responsiva en cualquier dispositivo iOS, así que mediante el uso de constrains, me enfoqué en crear una interfaz de usuario intuitiva y sencilla, utilizando celdas customizadas para mostrar los datos de los pokemon.

iOS Calculator

En este proyecto, he creado una réplica de la calculadora de iOS utilizando el patrón MVC. Comencé desarrollando la interfaz sin utilizar stacks, solo con constrains, como parte de mi proceso de aprendizaje. Una vez que consolidé mis conocimientos, implementé el uso de stacks para agrupar los botones de manera más eficiente. Después de diseñar completamente la interfaz, me enfoqué en dar funcionalidad a cada botón y codificar las operaciones que la calculadora permite realizar. A través de este proyecto, adquirí experiencia en el diseño de interfaces, el patrón MVC y la implementación de lógica para una aplicación práctica como una calculadora.
